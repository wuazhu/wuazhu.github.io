面向对象复习

基本类型是值的复制

对象类型不仅是值的复制, 而且也是引用的传递

比较也同上

原型: 如果一个对象下有一个方法, 如果每次 new 对象之后, 就会在内存中生成多份, 这样就会极大占用内存, 原型就是为了让对象下的方法或者属性在内存中只存在一份. 相当于 css 中的 class

如上方法, 只能在arr对象下调用自身的sum方法, 如果重新声明一个数组, 想调用 sum方法就不行, 那么原型就是为了解决这个问题.

原型链: 如果一个对象有一个属性,那么会优先找到这个属性, 如果没有就沿着原型链往内找,直到找到为止, 或者找到 Object.proto

这个原型对象就相当于 css 里的id是最优先级, 一旦有值就返回该值, 而原型的 prototype 相当于 class, 而Object.__proto__相当于[name=’xx’]属性选择器

实例对象.hasOwnProperty(‘属性名’)
检测对象自身是否拥有某个属性

实例对象.constructor
查看实例对象的构造函数

实例对象 instanceof 构造函数
检测对象与构造函数在原型链上是否有关系, 所有对象的最外层构造函数都是 Object
